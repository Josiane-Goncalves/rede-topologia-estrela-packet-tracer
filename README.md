# Projeto de Rede - Topologia Estrela

## 📌 Descrição

Projeto criado no Cisco Packet Tracer simulando a rede de comunicação da equipe de produção de um show.

## 🚀 Tecnologias

- Cisco Packet Tracer
- 1 Switch
- 4 Computadores
- Topologia estrela

## 🧠 Estrutura da Rede

PC1 - 192.168.1.1  
PC2 - 192.168.1.2  
PC3 - 192.168.1.3  
PC4 - 192.168.1.4
